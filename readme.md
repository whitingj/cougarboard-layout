## Motivation
Cougarboard recently redid their homepage layout de-emphasizing the message board content. The worst crime, IMO, is that to see the post in each category you have to click more and when you press the back button, you have to press it again. Additionally the message board content is off to the side.

This user javascript, re-arranges cougarboards layout and puts the message board content more prominently and expands all the different categories. I considered putting the message board first but decided to mirror the mobile layout. This can also be used on mobile to automatically expand all the categories.

## How to install:
* First install TamperMonkey:
  * [TamperMonkey Chrome Web Store](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en&pli=1)
* Then navigate to the raw cougarboard.user.js scripts and choose install:
  * [cougarboard.user.js RAW user.js](https://github.com/whitingj/cougarboard-layout/raw/refs/heads/master/cougarboard.user.js)

Alternatively you can just copy and paste cougarboard.user.js into a new TamperMonkey script and manage it yourself.
